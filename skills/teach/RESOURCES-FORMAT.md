# RESOURCES.md 格式

`RESOURCES.md` 是针对此主题精心筛选的可信来源集。解释性内容的知识应来自这里，而非参数化猜测。智慧来自这里列出的社区。

## 结构

```md
# {主题} 资源

## 知识

- [书籍：《力量训练的科学与实践》— Zatsiorsky & Kraemer](https://example.com)
  关于训练计划和适应的基础读物。用于：与周期化、恢复、强度区间相关的任何内容。
- [文章："我应该训练多少？"— Greg Nuckols (Stronger By Science)](https://example.com)
  基于证据的训练量标志性综述。用于：每肌肉组周组数目标。

## 智慧（社区）

- [r/weightroom](https://reddit.com/r/weightroom)
  高信号 subreddit，严格反民科。用于：训练计划评估、平台期排查。
- 本地：{健身房名称} 周二力量课
  用于：动作的实时教练反馈。
```

## 规则

- **仅限高信任来源。** 优先选择一手来源、公认的专家、同行评审的工作和具有强审核的社区。如果某个资源是伪装成教育的营销，删除它。
- **每一条目都要注释。** 三个月后，一个裸链接毫无用处。添加一行说明：它涵盖了什么以及何时使用它。
- **按知识/智慧分组。** 反映 [SKILL.md](./SKILL.md) 中的理念。一个资源只出现在一组中是完全可以的。
- **明确标记缺口。** 如果任务需要的某个领域没有好的资源，写一个 `## 缺口` 部分列出缺失的内容。这将驱动未来的搜索。
- **无情修剪。** 被证明是错误的、浅薄的或偏离任务的资源应该被移除，而不是被掩埋。五个精选的来源胜于三十个平庸的来源。
- **记录社区偏好。** 如果用户选择不加入社区，在此注明，以便未来学习不再重复推荐。
