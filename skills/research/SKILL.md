---
name: research
description: 针对高可信度一级来源调查一个问题，并将发现作为 Markdown 文件捕获到仓库中。当用户想要调研一个主题、收集文档或 API 事实、或将阅读工作委托给后台 agent 时使用。
---

启动一个**后台 agent** 来执行研究，让你在它阅读时继续工作。

它的工作：

1. 针对**一级来源**调查问题——官方文档、源代码、规格说明、第一方 API——而不是它们的二手解读。将每个主张追溯至拥有它的来源。
2. 将发现写入单个 Markdown 文件，引用每个主张的来源。
3. 保存在仓库已存放此类笔记的地方；匹配现有约定，如果没有，放在某个合理的位置并说明位置。
